# random_nickname
`random_nickname` is a Dart package that provides random nickname.

## Features
### Available Language Data
- English
    - Noun
        - animal
- Korean
    - Noun
        - animal
    - Adjective
        - emotion

## Getting started
### Install package
```bash
flutter pub add random_nickname
```
